int calc(int* arr, int size)
{
  int ans = 0;
  for (int i = 1; i < size; i++)
  {
    ans += arr[i] - arr[i-1];
  }
  return ans;
}

for от i до n-1

Асимптотика по времени будет линейной О(n), т.к. при увеличении массива в n раз количество шагов для вычислений также увеличится в n раз.
Асимптотика памяти будет константной О(1), т.к. для решения поставленной задачи мы используем только одну переменную sum.
